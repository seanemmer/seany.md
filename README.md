# seany.md

A recommended stack and set of coding rules based on my experience using Claude Code in the field. Drop the `CLAUDE.md` into your project and adapt as needed.

This file is intentionally minimal — just the highest-signal recommendations for getting started.

Feedback encouraged, stars appreciated 🤠

## Additional Recommendations

- **Turn on Agent Teams** — Enable `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS` in your `.claude/settings.json` for parallel agent workflows.
- **Install Official Anthropic Plugins** — Use `/plugin` to browse and enable the official plugin set.
- **Start in Plan Mode** — Use `/plan` to enter planning mode and align on approach before writing code.
- **Design Plans with Intermediate Steps** — Break work into incremental steps with validation between each, rather than all-or-nothing implementations.
